Promote digital sovereignty
Enable African-led innovation in harm reduction and public health

**System Architecture Overview**
**Frontend:** Web-based user interface for clinicians, nurses, pyschologists, HRIOs and pharmacists
**Backend:** PHP & MySQL
**Integration Layer:** Custom RESTful API
**Hardware:** MasterFlex¬Æ Methadone Dispensing Pump
**Deployment:** On-premise (clinic-based) or hybrid setups
